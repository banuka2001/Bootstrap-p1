My First Bootstrap Website  - very simple one

This is my first website built using Bootstrap. Instead of using online CDN links, I have downloaded and added all Bootstrap files locally.  

## Features  
- Built with Bootstrap for responsive design  
- Uses locally stored Bootstrap files  
- Simple and clean layout

## How to Run  
1. Clone the repository:  
   ```bash
   git clone <repo-url>
   ```  
2. Open the `index.html` file in a browser.
